# Automated-Attendance-system
Automated attendance system using opencv and face-recognition library

Prerequisites:
pip install opencv

Install Visual Studio Installer . Install Build tools for C++, where in CMake is included.

Install the dlib library using pip install dlib , CMake is required to install this.

pip install face-recognition


Using conda

conda create --name face_recog python=3.8

pip install face-recognition

python attendance.py
